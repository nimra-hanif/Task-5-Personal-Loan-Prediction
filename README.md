Task Objective: The objective of this project is to predict whether a bank customer will accept a personal loan offer or not. The aim is to help the bank identify potential customers and improve the effectiveness of its marketing campaigns by focusing on the right audience. Approach: In this project, the Bank Marketing dataset was used. First, the data was explored to understand customer features such as age, job, marital status, and financial details. After that, the data was cleaned by handling missing values, removing duplicates, and converting categorical variables into numerical form. Exploratory Data Analysis (EDA) was performed to identify patterns and trends in the data. Then, a Logistic Regression model was trained using the processed dataset. The model was evaluated using a test dataset to measure its performance. Results and Insights: The model achieved an accuracy of approximately 79%, which means it correctly predicted loan acceptance in most cases. The results showed that customers with higher financial stability, middle-aged individuals, and those in professional job roles are more likely to accept the loan offer. The confusion matrix revealed that while the model performs well overall, it still makes some incorrect predictions, especially missing a few potential customers who would accept the loan. Overall, the model provides useful insights for banks to target the right customers and improve marketing efficiency.
